BCA Project






Online Fabrication Shopping Website

In my BCA, I completed a Fabrication Website project as a solo. This website allows users to browse and shop various fabrication products, with the added functionality to add items to a cart for easy purchasing. I designed and developed a fully functional, user-friendly platform using HTML, CSS, JavaScript, Bootstrap, and PHP to ensure a responsive and interactive experience.

Handling all aspects of the project independently helped me improve my web development skills and gain practical experience in building an e-commerce style website tailored to fabrication services.

Frontend Technologies Used



HTML


HTML is used to create the basic structure of the website, including pages, forms, and content layout.

CSS


CSS is used to style the website, add colors, fonts, and make the layout responsive.

JavaScript


JavaScript is used to add interactivity such as sliders, form validation, and dynamic content updates.

Bootsrap


Bootstrap is used to create responsive and mobile-friendly layouts quickly using ready-made components.

PHP


PHP is used for server-side processing, handling forms, and connecting the website to a database.

Backend Technologies Used

Backend

SQL(structured query language)

In the fabrication website , SQL serves as the central backend database technology responsible
for managing and organizing all website data efficiently. It stores comprehensive product information such as names, categories, prices, descriptions, and availability, ensuring users always see accurate and up-to-date content. SQL also manages user-related data including registration details, login credentials, and session-based cart information, providing a secure and smooth shopping experience.

Cart operations are handled dynamically, allowing real-time updates when products are added or modified. Additionally, SQL supports order management by maintaining detailed records of orders, quantities, and transaction details, which helps in tracking purchases and streamlining backend operations. Overall, SQL plays a vital role in integrating front-end interfaces with server-side functionality, making the fabrication shopping website fully functional, dynamic, and data-driven.
